### Creating Premium Bookings

1. **Access Admin Panel**
   - Navigate to `/premium_admin.html` in your browser
   - You must be logged in as an admin

2. **Select Matches**
   - Browse available matches in the "Create New" tab
   - Use the search box to filter matches by team or league
   - Click on market options (Home Win, Draw, Away Win) to add them to your selections
   - Selected markets will be highlighted in green

3. **Configure Booking**
   - Set the **Price** (default $250)
   - Add an optional **Description** (e.g., "VIP Weekend Picks", "Championship Special")
   - Set **Expires After** hours (how long the code is valid)

4. **Generate Code**
   - Click "Create Premium Booking"
   - A unique code will be generated (e.g., `PRM1A2B3C4`)
   - **Share this code with your customers**

5. **Manage Bookings**
   - Switch to "Manage Bookings" tab
   - View all created bookings with:
     - Number of purchases
     - Total revenue
     - Status (active/cancelled)
   - Cancel bookings if needed

## ğŸ‘¤ User Guide

### Purchasing Premium Bookings

1. **Access Premium Tab**
   - Log in to your account
   - Click the **Premium** tab (with crown icon)

2. **Enter Code**
   - Type the premium booking code you received (e.g., `PRM1A2B3C4`)
   - Click "Check Code"

3. **Preview Booking**
   - You'll see:
     - Number of selections (matches)
     - Total odds
     - Price ($250)
     - Description (if provided)
   - **Note:** Actual match selections are hidden until purchase

4. **Purchase**
   - Click "Purchase Booking"
   - Confirm the transaction
   - $250 will be deducted from your balance

5. **View Selections**
   - After purchase, all match selections are revealed
   - View them anytime in "My Purchased Bookings" section
   - Use the picks to place your bets
